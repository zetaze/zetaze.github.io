从今年年初的时候就下定决心要走上游戏设计这条道路，然而却总是懒惰，拖沓。趁着这次学校成立 VR 相关的小部落，做一个资源的整理，也算是督促自己，赶紧学点真知识，也别总做个嘴炮。

06-28-2017 更新内容

好久没更新博客了，这段时间其实也没干什么事情，装电脑装了一次又一次，SteamVR 的文档看了看，然后…就没有然后了吧。
不过还是有一些资源要分享的。

首先就是这本书 Unity 5.X从入门到精通

肥肠厚肥肠重的一本书，天天拿着这个出门真是要了我的小命。
不过书的内容十分丰富，配图很多，而且是全部彩色印刷，看起来也蛮舒服的。
书的内容十分详细，也很基础，很适合初学者接触 Unity 的时候阅读。
因为配图多，文字就显得并不是很多，所以说如果专心读的话应该还是很快就能读完的。
总之，这本书很推荐阅读。

还有关于 VR 的一些….
Unity 由于不能原生支持 HTC Vive ，所以需要一些组件的支持才可以。
SteamVR Plugin
就是这个插件。
里面有官方的文档，介绍了 VR 中一些名词，已经怎么在 Unity 里设置，配置 Vive。 不过是英文的，看呗，怕个锤子。

06-08-2017 更新内容

Unity 官方教程

这里收集的是 Unity 官方教程，很权威，文档相关的内容也随着软件的更新而及时更新，可以说是相当专业了。
缺点就是在有些网络情况下，打开这些网站的速度极慢，而且视频教程都存在于一个并不存在的网站上，所以还需要掌握科学上网的技能。
另外，所有的官方文档，教程都是 没有中文 的，但是据我观察，其实看懂这些所需要的英语水平并没有多高，一些难理解的词汇借助翻译软件也可以理解，视频也都配有字幕，也就没有那么难理解了。
其实这些网站稍微搜索一下就都可以得到，但还是做个汇总吧，起码显得文章不那么…短。

  • Unity 官方文档

https://docs.unity3d.com/Manual/
这个可以说是 Unity 的词典了，任何 Unity 的专业词汇都可以在这里找到，图文并茂，可以说是相当专业了。
在文档首页的左侧有各种分类，我们需要 Virtual Reality 也在其中

文档的右上角有搜索框，更方便查找。

同时也有离线文档可用，方便在断网的时候查找。

也算挖个小坑吧，如果我有很多时间的话，我会翻译我看过的文档，之前看过一个人翻译的 C# Language Specification 5.0 ，官方都没有，吊吊的。

  • Unity 官方教程

感觉这些大型的工具类软件都会有官方教程，以便于初学者更加容易地入手，也算是推广自己产品的一种方式吧，一个软件都没人会用，那岂不是非常尴尬。
https://unity3d.com/cn/learn
这个页面应该说是 Unity 学习的总页面，各种详细的教程都可以从这个页面进入，就包括上面说过的文档。
个人觉得更加实用的还是这个 教程 界面，从最基础的项目到各种主题，一应俱全。
这里的各种教程炒鸡多，就比如 这个页面 ，本来想截张图放进来发现并截不全
有时候自己点开就感觉贼有动力去学…

第三方教育资源

虽然 Unity 的官方教程已经肥肠棒了，但是由于全英文并且需要科学上网，一些国内的本地化做的更好的教程也受到很多人的追捧，在我在各大论坛，贴吧搜索过后，发现 siki 的教程很多人还蛮推荐的，在这里分享一下。

微信在上方，回复后会有各种免费的视频教程可以下载。
这里是 siki 公众号内的视频汇总，里面有百度云的下载链接，在这里贴出来。
个人感觉他的系列视频讲解的还是蛮细致的，但是由于软件的更新，一些视频已经“过时”，自己选择性的看看就好，个人还是更加推崇官方的内容的，各取精华嘛。
